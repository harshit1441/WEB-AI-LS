# WEB-AI-LS

Created by Harshit Agarwal.

This repository contains weekly web development assignments built with HTML, CSS, and JavaScript. Each week focuses on a different front-end concept and keeps the code organized in its own folder.

## Week 1: Login Page

Week 1 is a responsive login page created with HTML and CSS. The page uses a centered card layout, styled form controls, and a gradient background to create a polished sign-in experience.

What it includes:
- A clean login form with email and password fields
- Responsive layout that stays centered on the screen
- Styled button and hover effects
- A simple forgot-password link

Files:
- [Week1/index.html](Week1/index.html)
- [Week1/style.css](Week1/style.css)

## Week 2: Task Manager

Week 2 is a Task Manager built with HTML, CSS, and JavaScript. It lets users manage tasks directly in the browser and demonstrates dynamic DOM updates and basic local storage usage.

What it includes:
- Add new tasks to the list
- Validate user input before submission
- Prevent empty task submissions
- Show success and error notifications
- Edit existing tasks
- Delete individual tasks
- Clear all tasks at once
- Persist tasks using local storage

Files:
- [Week2/index.html](Week2/index.html)
- [Week2/style.css](Week2/style.css)
- [Week2/script.js](Week2/script.js)
